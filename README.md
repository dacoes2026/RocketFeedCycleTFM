# RocketFeedCycleTFM
Main scripts and auxiliary function used for the development of the thesis "Study of the feed cycle of rocket engines: applications". 
The main scripts are referred to as "maindefX.m" and there are in total 4 of them. In increasing order, they correspond to the feed cycles of the rocket engines "SE-21D" (1), the Space Shuttle Main Engine (2), "SE-12" (3) & "RD-0120". 
In order for these codes to function please download the open source codes NFP & HGS they work with with the following links: https://github.com/ManelSoria/INIST & https://github.com/ManelSoria/HGS.
